# Echor - A Rust Rewrite of GNU/BSD echo program

``` text
echor 0.1.0
    Minh Tran <minh@minhtrannhat.com>
Rust rewrite of echo

Usage: echor [OPTIONS] <TEXT>...
Arguments:
  <TEXT>...  Input text

Options:
  -n             Do not print newline
  -h, --help     Print help
  -V, --version  Print version
```

## Build

`cargo build`

## Test
`cargo test`
